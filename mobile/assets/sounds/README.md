# Sonidos del Scanner PlantCare

Para activar los efectos de sonido del scanner, agrega estos archivos:

## Archivos necesarios

1. **scanner-beep.mp3** - Beep repetitivo estilo sci-fi (~0.2-0.5s)
2. **scanner-complete.mp3** - Ding de completado (~0.5-1s)

## Cómo obtener sonidos libres

### Opción 1: Freesound.org (gratis con cuenta)
- https://freesound.org
- Buscar: "scanner beep", "sci-fi beep", "notification success"
- Descargar en formato MP3
- Licencia: Creative Commons (revisar atribución)

### Opción 2: Mixkit (gratis, sin cuenta)
- https://mixkit.co/free-sound-effects/
- Buscar efectos de "technology" o "notification"

### Opción 3: Sonidos de sistema
- Usar tonos cortos de iOS/Android como referencia
- Regrabar o generar con Audacity (gratis)

### Opción 4: Generar con Audacity
- Descargar Audacity (gratis): https://www.audacityteam.org/
- Generar → Tono: onda cuadrada, 800-1200Hz, 0.2s para beep
- Para complete: tono más melódico o "ding" ascendente

## Renombrar
Coloca los archivos aquí con exactamente estos nombres:
- `scanner-beep.mp3`
- `scanner-complete.mp3`

Si no existen, el scanner funcionará en silencio (sin errores).
