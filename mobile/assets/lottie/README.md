# Animaciones Lottie - PlantCare

Para la experiencia completa, descarga estos JSON desde LottieFiles y colócalos aquí:

| Archivo | Descripción | Buscar en lottiefiles.com |
|---------|-------------|---------------------------|
| `plant-happy.json` | Plantita bailando (protagonista) | "plant animation" |
| `camera-scan.json` | Cámara escaneando | "camera scanning" |
| `gallery.json` | Galería / fotos | "gallery photos" |
| `scan-lines.json` | Líneas de scanner | "scan" |
| `success.json` | Check completado | "success check" |
| `confetti.json` | Celebración | "confetti" |

Si los archivos no existen, la app usa animaciones con Reanimated + Emoji (funciona en Expo Go).
